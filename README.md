# svm-kernel-tutorial
## Overview
This project explores how different kernel functions affect Support Vector Machines (SVMs). It compares linear, polynomial and RBF kernels using a non-linear dataset.

## Dataset
The dataset used is `make_moons` from scikit-learn. It contains two curved classes and is useful for testing non-linear models.

## Models Used
- Linear kernel
- Polynomial kernel
- RBF kernel

## What this project shows
- How decision boundaries change with kernels
- Why linear models fail on non-linear data
- Why RBF performs best
- Effect of hyperparameters (C and gamma)

## How to run
1. Open Jupyter Notebook  
2. Open:
   notebook/svm_kernels_tutorial.ipynb  
3. Run all cells  

## Files in this project
- notebook → contains the code  
- report → contains the written report  
- figures → contains all plots
